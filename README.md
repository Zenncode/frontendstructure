# zenncode

`zenncode` is a frontend scaffolding CLI that creates a clean project structure with optional starter files and smart defaults.

## Install

```bash
npm i zenncode
```

## Run without install

```bash
npx zenncode init
```

## Usage

```bash
zenncode init
zenncode init --tailwind --ts --router
zenncode init --framework next --yes
zenncode init --interactive
```

## Available options

- `--tailwind` / `--no-tailwind`
- `--ts` / `--no-ts`
- `--router` / `--no-router`
- `--framework <react|vite|next>`
- `--install` / `--no-install`
- `--interactive`
- `--yes`
- `--help`
